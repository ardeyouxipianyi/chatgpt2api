# 更新日志

## v1.1.8

这一版主要整理了 v1.0 发布后围绕网页、画布、号池、部署和数据迁移做的一批改进。

### 新增

- 新增画布式图片创作页，支持提示词生成、图片编辑分支、节点复制、节点删除、整理布局、上游高亮、图片弹窗预览、画布保存/恢复。
- 新增画布反推提示词功能，可上传图片或使用画布图片生成反推提示词。
- 新增设置页管理员密码修改功能。
- 新增配置数据导入/导出功能，可选择系统配置、账号池、用户密钥、图片任务、画布项目、生成图片、日志、注册配置、CPA 配置、Sub2API 配置。
- 新增账号一键刷新进度展示与后台刷新任务。
- 新增大批量账号导入的后台刷新流程，减少导入等待时间。
- 新增图片失败自动换号、空结果重试、额度耗尽识别与账号冷却能力。

### 优化

- 优化画布任务处理逻辑，画布中已有任务处理中时仍可继续创建其他任务。
- 优化画布在不同宽度下的布局，避免底部编辑框被挤出或不可编辑。
- 优化图片生成等待策略：远端未确认进入生图任务时快速切号，远端确认后结果长时间不出现时按卡住处理。
- 优化账号类型识别，兼容更多远端账号计划字段。
- 优化 Windows 本地运行方式，统一网页和 OpenAI 兼容接口到 `3000` 端口，内部 Next 服务仅绑定本机。
- 优化 Cherry Studio / OpenAI 兼容接口图片结果返回与错误处理。

### 修复

- 修复图片生成成功但客户端拿不到图片时的错误处理。
- 修复画布新增节点生成位置可能覆盖已有节点的问题。
- 修复设置页部分大类设置折叠/隐藏体验。
- 修复反推提示词取消、默认提示词保存和显示问题。
- 修复账号额度文本提示未及时标记限流的问题。

### Windows 绿色版

- 继续支持 Windows 绿色版结构，用户解压后运行 `start.bat`，访问 `http://localhost:3000`。
- Cherry Studio / OpenAI 兼容接口地址为 `http://localhost:3000/v1`。
